Como usar o comando snoop no Solaris

Para que server o comando SNOOP

Este comando é usado para realizar captura de pacotes nas interfaces do servidor, auxiliando nas analises e tshoot do dia dia de um profissional de TI.

Clique aqui para maiores infos

Syntax basica do comando SNOOP

Um exemplo de syntax base do snoop é: comando(snoop) -d(interface que sera capturado os pacotes) filtros(como IP,porta, drop de pacotes)

Syntax avançada do comando SNOOP

Um exemplo de syntax avançada do snoop é: comando(snoop) -v (habilitar o verbose) -o (direcionar a saida para um arquivo) -d(interface que sera capturado os pacotes) filtros(como IP,porta, drop de pacotes) (uso de src ou dst para especificar a analise)
